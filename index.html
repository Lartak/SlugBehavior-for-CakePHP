SlugBehavior-for-CakePHP

Adapté pour CakePHP version 2.4

Voir sur GitHub English
Téléchargements:
  • ZIP

    Présentation

    Ce comportement (Behavior) automatise le développement de slug lors d'une sauvegarde, soit au moment de la création de l'enregistrement soit lors de la modification.

    Installation

    Pour l'utiliser, importer le fichier dans votre dossier d'application CakePHP Behavior (app/Model/Behavior).

    Puis ajouter dans le modèle que vous souhaitez utiliser:

    public $actsAs = array('Slug');

    Les options par défaut du Behavior sont :

    • Pour le champ source : name
    • Pour le champ de sauvegarde du slug : slug
    • Le séparateur utilisé : -

    Pour modifier une de ses options ou toutes, voir dans la partie utilisation.

    Utilisation

    Vous pouvez modifier les options lorsque vous appelez le Behavior dans votre modèle comme ceci:

    `public $actsAs  = array(
  'Slug'  =>  array(
    'fieldName' =>  'title',
    'fieldSlug' =>  'slug2',
    'separator' =>  '_'
    )
  );`

    Vous n'avez rien d'autre à faire, au cours de chaque sauvegarde de votre modèle, le Behavior effectuera le slug.
